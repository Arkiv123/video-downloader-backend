## `01` &nbsp;WHAT THIS IS

**GOOGLY RANKS** runs a broadcast-styled feed puller. Drop a link from any platform, the deck reads every resolution and audio track the source carries, and lays it out on a flap board. Pick a grade. Hit **ON AIR**. The clip is yours.

No watermarks. No re-compression. No "download unavailable" walls.

## `03` &nbsp;WHY DONATIONS EXIST

<div align="center">

This entire deck runs on **free-tier infrastructure** — that's how it stays free for everyone. Free tiers cap out fast though: **up to 4K works fine on what we've got.** Pushing **8K** needs a real server upgrade — more compute, more bandwidth, none of it free.

**Donating is not required to use the deck.** 4K stays free either way. If you want to help push it further, it's appreciated — entirely your call.
