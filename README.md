## `01` &nbsp;WHAT THIS IS

**RIPDECK** is a self-hosted, broadcast-styled personal archive tool. Paste a link from any platform — the deck reads every resolution and audio track your source carries and lays it out on a flap board. Pick a quality. Hit **ON AIR**. Grab the file.

Built for creators who need their own uploaded content back — no watermarks, no re-compression, no "download unavailable" walls.

<br/>

## `02` &nbsp;THE STACK

<div align="center">

| LAYER | TECH |
|---|---|
| **Frontend** | Cloudflare Pages · Vanilla JS · Custom broadcast UI |
| **Backend** | FastAPI · Python 3.11 · Dockerized |
| **Engine** | [`yt-dlp`](https://github.com/yt-dlp/yt-dlp) + `aria2c` (16-thread parallel pulls) |
| **Hosting** | 100% free tier — Cloudflare Pages + Render |

</div>

<br/>

## `03` &nbsp;FEATURES

```
▸ Multi-platform support        YouTube · TikTok · Instagram · X · Facebook · Reddit · Vimeo · Twitch
▸ Full quality board             Every resolution + audio bitrate, no guessing
▸ Max-speed pulls                 16-connection aria2c downloads
▸ Audio extraction                Clean MP3 rips from any video source
▸ Zero cost to run                Free-tier hosted, no subscriptions
```

<br/>

## `04` &nbsp;RUN IT YOURSELF

```bash
git clone https://github.com/YOUR_USERNAME/googly-ranks-ripdeck.git
cd googly-ranks-ripdeck

# backend
pip install -r requirements.txt
uvicorn backend:app --reload

# frontend — just open index.html, or deploy the folder to Cloudflare Pages
```
